# 브라우저로 공유해서 사용하는 방법

직원 PC에서 start-cafe-app.bat 파일을 더블클릭하면 주문 앱 서버가 켜집니다.

직원 PC 주소:
http://localhost:3000/

같은 Wi-Fi에 연결된 휴대폰/다른 PC 공유 주소:
http://172.30.1.41:3000/

사용 순서:
1. 직원 PC에서 start-cafe-app.bat을 실행합니다.
2. 직원 PC 브라우저에서 http://localhost:3000/ 을 엽니다.
3. 손님에게 http://172.30.1.41:3000/ 주소를 공유합니다.
4. 직원 화면은 비밀번호 cafe1234로 들어갑니다.

주의:
- 직원 PC와 손님 기기가 같은 Wi-Fi에 있어야 합니다.
- 직원 PC가 꺼지거나 실행 창을 닫으면 주문 페이지도 멈춥니다.
- 학교/기관 Wi-Fi에서 접속이 안 되면 Windows 방화벽에서 Node.js 허용이 필요할 수 있습니다.

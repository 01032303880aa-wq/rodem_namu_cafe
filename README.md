# 로뎀나무 CAFE 주문 앱

이 버전은 여러 기기에서 같은 주문 목록을 볼 수 있는 서버형 앱입니다.

## 같은 Wi-Fi에서 쓰기

1. 카페 직원 PC에서 이 폴더를 엽니다.
2. 터미널에서 아래 명령을 실행합니다.

   node server.js

3. 직원 PC에서는 아래 주소로 접속합니다.

   http://localhost:3000

4. 손님 휴대폰이나 다른 기기는 직원 PC의 IP 주소로 접속합니다.

   http://직원PC_IP주소:3000

직원 화면 비밀번호 기본값은 cafe1234 입니다.

## 어디서든 접속하려면

Node.js 앱을 올릴 수 있는 호스팅 서비스에 이 폴더를 업로드하면 됩니다.
시작 명령은 아래와 같습니다.

node server.js

환경 변수 STAFF_PASSWORD를 설정하면 직원 비밀번호를 바꿀 수 있습니다.
